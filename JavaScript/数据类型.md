## 原始数据类型: number, string, boolean, undefined, null, object

#### 基本类型(简单类型), 值类型: 

number, string, boolean

#### 复杂类型(引用类型): 

object  (其实null也是)

#### 空类型: 

undefined, null

- 值类型的值在那一块空间中存储? 
  - 栈中存储
- 引用类型的值在哪一块空间中存储?
  - 对象在堆上存储, 地址在栈上存储

##### 值类型之间传递的是值; 引用类型之间传递地是地址(引用)
